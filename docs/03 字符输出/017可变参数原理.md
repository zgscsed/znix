# 可变参数原理

```c++
int printf(const char* format, ...);
```

- `va_list`：保存可变参数指针
- `va_start`：启用可变参数
- `va_arg`：获取下一个参数
- `va_end`：结束可变参数


```c++
void test_args(int cnt, ...)
{
    va_list args;
    va_start(args, cnt);
    int arg;
    
    while (cnt--)
    {
        arg = va_arg(args, int);
    }
    va_end(args);
}

test_args(5, 1, 0xaa, 5, 0x55, 10);
```
